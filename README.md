# SCU Foundation — Legal & Compliance

This repository contains the canonical legal, policy, and compliance documents of the SCU Foundation.  
These documents define the Foundation’s public‑facing legal commitments, disclosure processes, licensing terms, and institutional guarantees.

All documents in this repository are canonical, pre‑normative, and maintained by the SCU Foundation as part of its long‑term, neutral, public‑good stewardship.

The canonical repository index is:

- REPO: Repo‑Map — /REPO-MAP-vE1.1.md  
- REPO: Repo‑Map — /GRM-vE1.1.json  

---

## Purpose

The Legal & Compliance corpus ensures that the SCU ecosystem operates with:

- transparency  
- institutional neutrality  
- public‑sector readiness  
- clear rights and responsibilities  
- open licensing  
- responsible disclosure pathways  
- long‑term trust and accountability  

These documents apply to all SCU Foundation websites, publications, standards, and public surfaces.

## Scope of This Repository

This repository includes:

- legal policies  
- compliance documents  
- disclosure procedures  
- licensing and attribution rules  
- institutional guarantees  
- public‑sector alignment statements (legal surface only)  

This repository does **not** include:

- governance documents (see REPO: Governance — /)  
- global‑rules documents (see REPO: Governance — /global-rules/)  
- standards or specifications (see REPO: Standards — / and REPO: Specifications — /)  
- research outputs (LRI)  
- website source code (`scu-foundation/`)
