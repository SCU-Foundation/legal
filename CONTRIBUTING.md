# Contributing — Legal Repository

Thank you for your interest in contributing to the SCU ecosystem.

The canonical repository index is:

- REPO: Repo‑Map — /REPO-MAP-vE1.1.md  
- REPO: Repo‑Map — /GRM-vE1.1.json  

## Scope of This Repository

This repository contains the **legal and compliance surfaces** of the SCU ecosystem, including:

- privacy policy  
- terms of use  
- licensing  
- attribution  
- safety and security disclosures  

These documents define the legal framing for all SCU Foundation programs and public surfaces.

## Contribution Rules

All changes MUST follow:

- Update‑Sequence Charter  
  - REPO: Governance — /global-rules/USC-vE1.1.md  
- Canonical Document Responsibilities  
  - REPO: Governance — /global-rules/CDR-vE1.1.md  
- Naming & Terminology Standard  
  - REPO: Governance — /global-rules/NTS-vE1.1.md  
- Versioning Rules  
  - REPO: Governance — /global-rules/VR-vE1.1.md  
- Safety Charter  
  - REPO: Governance — /global-rules/SC-vE1.1.md  

## Process

1. Open an issue describing the proposed change.  
2. Reference relevant canonical documents.  
3. Submit a pull request linked to the issue.  
4. Await review from SCU Foundation maintainers.  

Legal documents MUST remain consistent with the Governance‑Root corpus and the SCU Foundation’s institutional framing.
